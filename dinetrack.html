DineTrack

Designed to bring simplicity to every step of the dining experience, DineTrack seamlessly integrates data management with interactive map views, making it effortless to find, save, and share your favorite restaurants. By combining robust functionality with an intuitive interface, the app transforms how users explore local eateries, discover new tastes, and keep track of must-visit spots.

Watch Video Github Repo

Technologies Used

Java

Core language ensuring reliability and performance for Android functionality.

SQLite

Local database to store user data securely and efficiently.

Google Maps

Integrated map views for real-time location insights.

Android Studio

IDE for designing UI, debugging, and performance optimization.

Key Features

Restaurant Data Management

Add, edit, and remove entries with name, address, tags, rating, and more.

Advanced Search & Filtering

Find the perfect spot using multiple filters (ratings, tags, distance).

Map Integration

Display restaurants on Google Maps with markers and navigation directions.

Share & Collaboration

Share listings with friends or export data for group planning.

Project Impact

DineTrack streamlines dining decisions by centralizing restaurant information in a single, intuitive app. The integration of Java, SQLite, and Google Maps fosters a robust, scalable solution for real-time searching and filtering. By offering advanced data management and interactive map features, DineTrack demonstrates a production-ready Android experience—ideal for everyday use or future commercial expansions.
