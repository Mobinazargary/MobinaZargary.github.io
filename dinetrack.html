DineTrack

Designed to bring simplicity to every step of the dining experience, DineTrack seamlessly integrates data management with interactive map views, making it effortless to find, save, and share your favorite restaurants. By combining robust functionality with an intuitive interface, the app transforms how users explore local eateries, discover new tastes, and keep track of must-visit spots.

Technologies Used

Java

Core language ensuring reliability and performance for Android functionality.

SQLite

Local database to store user data (restaurants, ratings, tags) securely and efficiently.

Google Maps

Integrated map views for real-time location insights and turn-by-turn navigation.

Android Studio

Comprehensive IDE for designing the UI, debugging, and performance optimization.
