Technologies Used

React.js

Frontend framework for building dynamic, component-based user interfaces.

OpenWeatherMap API

Fetching real-time weather data, including current conditions & forecasts.

Custom CSS

Responsive styling for a seamless experience across various devices.

Weather Icons

Icon sets from OpenWeatherMap for visual representation of conditions.

Key Features

Real-Time Weather

Fetches current weather data (temperature, humidity, wind, etc.) for any city via the OpenWeatherMap API, ensuring accurate results.

City Search with Debounce

Users can type their city name, and a debounce function prevents excessive API calls, ensuring smooth performance.

5-Day Forecast

Displays a multi-day outlook for upcoming weather conditions, helping users plan ahead.

Responsive UI

Designed to adapt to different screen sizes, ensuring a seamless experience on mobile, tablet, and desktop.
