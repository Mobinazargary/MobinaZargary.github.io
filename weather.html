React Weather App

A ReactJS-based weather application that provides current conditions and a 5-day forecast for any searched city. By leveraging the OpenWeatherMap API, it delivers accurate data on temperature, humidity, wind, and more—helping users plan their day with ease.

Watch Video Github Repo

Technologies Used

React.js

Frontend framework for building dynamic user interfaces.

OpenWeatherMap API

Fetches real-time weather data and forecasts.

Custom CSS

Responsive styling for a seamless cross‑device experience.

Weather Icons

Icon sets from OpenWeatherMap for visual condition cues.

Key Features

Real-Time Weather

Displays current weather data (temp, humidity, wind, etc.).

City Search with Debounce

Prevents excessive API calls for smooth performance.

5-Day Forecast

Shows multi-day outlook to help you plan ahead.

Responsive UI

Adapts seamlessly to mobile, tablet, and desktop.
